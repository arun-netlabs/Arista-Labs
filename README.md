# Arista Labs

A collection of hands-on Arista EOS, EVPN, MPLS, Segment Routing, and Data Center networking labs built using Containerlab.

## About

Hi, I'm Arun Ganesan.

This repository contains networking labs, configurations, packet-flow analysis, and technical articles that I use for learning, testing, and documenting modern service provider and data center technologies.

The goal is to provide simple, reproducible lab environments that can be deployed quickly and used for self-study or demonstrations.

## Available Labs

### EVPN-VPWS-Lab

* EVPN VPWS over SR-MPLS
* BGP EVPN Route Type-1 analysis
* Control-plane and data-plane verification

### L2-EVPN-Lab

* EVPN Layer-2 services
* MAC learning and advertisement
* BUM traffic handling

### L2-ISIS-Lab

* ISIS adjacency formation
* SRGB and Prefix-SID allocation
* SR-MPLS forwarding

### LDP-VPLS

* Traditional MPLS LDP signaling
* VPLS service deployment
* Packet forwarding analysis
